# Getting Started

### If you want to contribute to this project, follow these steps:

## 1. Clone the Repository:

```bash
git clone https://github.com/tahmid0111/Mediuum_Front_end.git
```
## 2. Install Dependencies:

```bash
cd Mediuum_Front_end
npm install
```

## 5. Start the Server:

```bash
npm run dev
```
